# Anki CLI

Небольшое консольное приложение для ведения словаря карточек и тренировки немецких слов.

## Возможности

- добавление, просмотр, редактирование и удаление карточек;
- отметки карточек: `important` и `muted`;
- тренировка слов в игровом режиме;
- поиск по словарю;
- импорт и экспорт карточек в CSV;
- слово дня из списка `top_1000_german_words.csv`;
- получение данных с verbformen.com для слова дня;
- опциональный перевод примеров через DeepL.

## Требования

- Python `3.10+`;
- `uv` или другой менеджер зависимостей Python;
- интернет для функции слова дня с verbformen.com;
- DeepL API key только для команды `!deepl` в игре.

## Установка

```bash
git clone <repo-url>
cd anki
uv sync
```

Если `uv` не используется:

```bash
python -m venv .venv
source .venv/bin/activate
pip install colorama thefuzz deepl dotenv
```

## Настройка DeepL

Для перевода примеров создайте файл `.env` в корне проекта:

```env
DEEPL_API_KEY=your_deepl_api_key
```

Без этого ключа приложение работает, но команда `!deepl` покажет сообщение об ошибке.

## Запуск

```bash
uv run python main.py
```

Или, если окружение уже активировано:

```bash
python main.py
```

При запуске приложение:

1. создаёт SQLite-базу, если её ещё нет;
2. показывает слово дня;
3. открывает интерактивный prompt `Anki>`.

## Где хранятся данные

Приложение хранит пользовательские данные в директории:

```text
~/.anki/
```

Основные файлы:

- `~/.anki/words.sqlite3` — база карточек;
- `~/.anki/daily_word.json` — сохранённое слово дня;
- `~/.anki/*.csv` — файлы для импорта/экспорта.

## Команды

| Команда | Описание |
| --- | --- |
| `help` | Показать список команд |
| `add` | Добавить новую карточку |
| `all` | Показать все карточки |
| `show [id]` | Показать карточку по ID |
| `edit [id]` | Изменить карточку по ID |
| `remove [id]` | Удалить карточку по ID |
| `remove 1,3,5-8` | Удалить несколько карточек |
| `find` | Найти карточки по слову |
| `important` | Показать важные карточки |
| `muted` | Показать muted-карточки |
| `play` | Запустить тренировку |
| `import` | Импортировать CSV из `~/.anki` |
| `export` | Экспортировать словарь в CSV |
| `csv` | Показать CSV-файлы в `~/.anki` |
| `reword` | Выбрать новое слово дня |
| `clear` | Очистить экран |
| `quit` | Выйти из приложения |

## Формат карточки

Карточка содержит:

- `word` — слово;
- `description` — перевод или описание;
- `example` — пример использования;
- `isImportant` — важная карточка;
- `isMuted` — скрытая карточка.

При добавлении или редактировании отметку можно указать так:

- `+`, `i`, `important`, `y`, `1` — important;
- `-`, `m`, `muted` — muted;
- `0`, `n`, `normal`, `none` — обычная карточка;
- пустой ввод — оставить значение по умолчанию или текущее значение.

## Игровой режим

Запуск:

```text
Anki> play
```

Перед игрой нужно выбрать:

1. режим вопроса:
   - `word -> translate -> example`;
   - `example -> translate -> word`;
2. порядок карточек:
   - случайный;
   - от новых к старым;
   - от старых к новым;
3. набор карточек:
   - все, кроме muted;
   - только important;
   - только muted;
   - все карточки.

Команды внутри игры:

| Команда | Действие |
| --- | --- |
| `!stop` | Завершить игру |
| `!swap` | Показать пример или слово |
| `!deepl` | Перевести пример через DeepL |
| `!mute` | Пометить текущую карточку как muted |
| `!important` | Пометить текущую карточку как important |
| `+` | Засчитать ответ и пометить карточку как muted |
| `-` | Пометить карточку как important после ошибки |

Ответ засчитывается как правильный при точном совпадении с переводом. Также используется fuzzy-проверка: небольшие опечатки могут быть засчитаны.

## CSV

Экспорт создаёт файл в `~/.anki` с разделителем `;`.

Формат строки:

```csv
word;description;example
```

Импорт читает CSV-файлы из `~/.anki`. Карточки с уже существующим словом пропускаются с сообщением об ошибке.

## Пример использования

```text
Anki> add
Enter a word: gehen
Enter a description: идти, ходить
Enter an example: Ich gehe nach Hause.
Card mark (+ important, - muted, Enter normal):

Anki> play
Anki> export
Anki> quit
```
